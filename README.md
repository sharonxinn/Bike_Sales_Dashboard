# Bike_Sales_Dashboard
Developed an interactive bike sales dashboard using Microsoft Excel to analyze customer purchasing behavior and identify key sales trends.

# Skill Performed

-Performed data cleaning (removed duplicates, standardized formats, handled missing values
-Conducted exploratory data analysis to uncover patterns in customer demographics and purchasing decisions
-Built pivot tables to summarize sales metrics by income, age group, gender and region
-Built visualizations (bar charts, line charts, slicers) to present insights clearly
-Created a dynamic dashboard for easy filtering and real-time analysis

# Key Insights 

-Identified midddle age group is most likely to purchase bikes
-Analyzed impact of commute distance on purchase decisions, distance 10+ miles away is most likely to purchase bikes
-Observed trends across income and region

# Tools Used
Microsoft Excel (Pivot Tables, Charts, Slicers, Data Cleaning)
